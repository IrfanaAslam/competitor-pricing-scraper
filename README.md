# Competitor Pricing & Stock Monitoring Scraper

A Python-based web scraping project that collects competitor pricing and stock availability data from multiple online sources.

## Features
- Scrapes product name, price, and availability
- Supports multiple competitor websites
- Stores results in CSV format
- Modular and scalable design
- Easy to extend for new websites

## Tech Stack
- Python
- Requests
- BeautifulSoup
- Pandas
